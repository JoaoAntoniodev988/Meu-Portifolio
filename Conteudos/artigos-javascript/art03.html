Manipulação de Dados

Manipular dados significa alterar o seu estado ou extrair informações úteis deles. Cada tipo de dado tem seus próprios "superpoderes" (métodos).

Manipulando Strings (Textos)

  • Descobrir o tamanho (.length): Conta quantos caracteres o texto tem.
  • Mudar a caixa (.toUpperCase() / .toLowerCase()): Deixar tudo em maiúsculo ou minúsculo.
  • Cortar pedaços (.slice()): Extrair apenas uma parte do texto.

JavaScript

const frase = "JavaScript é incrível!";

console.log(frase.length); // 22 (contando os espaços)

console.log(frase.toUpperCase()); // "JAVASCRIPT É INCRÍVEL!"

Manipulando Números (Objeto Math)

Para contas complexas ou ajustes, usamos o objeto nativo Math.

  • Arredondar: Math.round() (arredonda para o mais próximo)
  • Math.ceil() (para cima) ou Math.floor() (para baixo).
  • Gerar número aleatório: Math.random().

JavaScript

const nota = 7.6;

console.log(Math.round(nota)); // 8

console.log(Math.floor(nota)); // 7

Conversão de Dados (Type Casting / Coercion)

Às vezes, você recebe um dado em um formato (como um texto vindo de um formulário), mas precisa dele em outro formato (como um número para fazer uma conta). No JavaScript, existem duas formas de conversão:

Conversão Explícita (Você força a mudança)

É quando você usa comandos da linguagem para transformar o dado conscientemente. Para Número: Usa-se Number(), parseInt() (para inteiros) ou parseFloat() (para decimais). Para Texto: Usa-se String() ou o método .toString().

JavaScript

const textoIdade = "25";

const idadeNumero = Number(textoIdade); // Transforma em número real: 25

const numero = 100;

const numeroTexto = String(numero); // Transforma em texto: "100"

Conversão Implícita (O JavaScript tenta adivinhar)

O JavaScript é uma linguagem "compreensiva" até demais. Se você tentar somar um número com um texto, em vez de dar erro, ele converte o número em texto e junta os dois.

JavaScript

const resultado = "5" + 2;

console.log(resultado); // Vai exibir: "52" (Texto), e não 7!

Formatação de Dados

Formatar é preparar o texto para que ele seja exibido de forma bonita e legível para o usuário final.

Formatação de String

  • O Jeito Antigo: Concatenação. Juntar textos e variáveis usando o sinal de +. Costuma ficar confuso muito rápido.

    JavaScript

    const nome = "Carlos";

    const tecnologia = "React";

    console.log("O aluno " + nome + " está estudando " + tecnologia + ".");

  • O Jeito Moderno: Template Strings (ES6). Usamos as crases (` `) e colocamos as variáveis dentro de ${}. O código fica muito mais limpo e aceita quebra de linhas.

    JavaScript

    const nome = "Carlos";

    const tecnologia = "React"; // Muito mais fácil de ler e manter!

    console.log(`O aluno ${nome} está estudando ${tecnologia}.`);

Formatação de Números

Mostrar números brutos no sistema pode parecer pouco profissional (ex: 1250.5). Precisamos formatá-los para a realidade do usuário, como moedas ou casas decimais fixas.

  • Fixar Casas Decimais (.toFixed()): Determina quantas casas vão aparecer após o ponto. Atenção: este método transforma o número em uma string.

    JavaScript

    const precoBruto = 19.99321;

    console.log(precoBruto.toFixed(2)); // "19.99"
  • Formatação de Moedas e Regiões (Intl.NumberFormat): A forma mais profissional e poderosa do JavaScript para formatar moedas, porcentagens e padrões de países automaticamente.

    JavaScript

    const salario = 5000.50;

    const formatadorReal = new Intl.NumberFormat('pt-BR', { style: 'currency', currency: 'BRL' });

    console.log(formatadorReal.
