Mestre do CSS Grid Layout

Compreenda o funcionamento estrutural em duas dimensões e experimente todas as propriedades de container em tempo real.

1. O Modelo Bidimensional (2D)

O CSS Grid Layout opera criando uma malha invisível de linhas horizontais e verticais. A grande diferença para o Flexbox é que no Grid você dita o comportamento das linhas e das colunas simultaneamente a partir do elemento pai.

A revolução da unidade fr: Esqueça porcentagens (33.33%). Com 1fr 1fr 1fr, o navegador divide o espaço disponível em três partes exatamente iguais de forma automática.

2. Todas as Propriedades de Container

display

Define o elemento como contêiner grid (grid ou inline-grid).

grid-template-columns

Define o número e o tamanho das colunas da grade.

grid-template-rows

Define o número e o tamanho das linhas da grade.

grid-template

Atalho para definir rows, columns e áreas em uma única declaração.

gap

Espaçamento uniforme entre linhas e colunas (row-gap + column-gap).

grid-auto-rows

Tamanho padrão de linhas criadas implicitamente (fora do template).

grid-auto-columns

Tamanho padrão de colunas criadas implicitamente.

align-items

Alinha o conteúdo de cada item no eixo vertical dentro da célula.

justify-items

Alinha o conteúdo de cada item no eixo horizontal dentro da célula.

align-content

Alinha toda a grade no eixo vertical quando há espaço sobrando.

justify-content

Alinha toda a grade no eixo horizontal quando há espaço sobrando.

place-items

Atalho para align-items + justify-items.

place-content

Atalho para align-content + justify-content.

repeat()

Função para repetir padrões: repeat(3, 1fr) ou repeat(auto-fit, minmax(150px, 1fr)).

unidade fr

Representa uma fração do espaço livre disponível no container.
